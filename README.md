🎧 Spotify Money Flow — Who Gets Paid When You Press Play?
📌 Project Overview

This project explores how money flows through the music streaming ecosystem using Spotify as a case study. It presents a web-based interactive experience that explains how revenue is distributed among artists, labels, and platforms.

The goal is to simplify a complex system into an engaging and easy-to-understand format through visuals and storytelling.

📁 Project Structure

Below is a summary of the key files and folders in this project:

/src
Contains the main source code for the application, including core logic and components used to build the website.
/public
Stores static assets such as images, icons, and other media files used in the website.
index.html
The main HTML file that serves as the entry point for the web application.
package.json
Lists project dependencies and scripts required to run and manage the project.
vite.config.js (if present)
Configuration file for the Vite development environment used to build and serve the project.
/components or /pages (if present)
Contains reusable UI components or structured sections of the web page.
/assets (if present)
Includes additional design elements such as SVGs, charts, or illustrations.
🚀 How to Run the Project

Follow these steps to access and use the final product:

Clone the repository

git clone https://github.com/thatemocoder/spotify.git
cd spotify

Install dependencies

npm install

(or use pnpm install if configured)

Run the development server

npm run dev
Open in browser
Visit the local URL shown in your terminal (usually http://localhost:5173/).
